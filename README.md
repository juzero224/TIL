## Heading(#)

* `#` 해시태그의 개수만큼 헤딩의 중요도가 결정됨
* `h1`~ `h6` 태그가 있음



## List (1,*,-)

### 순서가 있는 리스트

1. 하나
2. 둘
3. 셋
4. 넷!

* `숫자`+`.`으로 작성 가능

### 순서가 없는 리스트

* 이거
* 저거
* 그거
* 요거
* `*/-` + `space bar`으로 작성 가능.
